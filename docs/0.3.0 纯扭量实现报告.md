## 📊 架构规格

### 核心方程 (来自 docs)

```
dz/dt = (-z + W·tanh(z) + U·x + b) / τ(z)

其中:
  - z ∈ ℂⁿ 是复数状态
  - τ(z) = sigmoid(W_τ·|z|) + ε 是状态依赖时间常数
  - W_real, W_imag 分别处理实部和虚部
  - U 是输入投影
  - b 是偏置项
```

### 三种配置

| 配置 | Small | Medium | Large |
|------|-------|--------|-------|
| **hidden_dim** | 256 | 512 | 2048 |
| **n_layers** | 4 | 8 | 16 |
| **参数量** | 1.8M | 14.7M | 470M |
| **输入** | 连续值 | 连续值 | 连续值 |

## 🎯 与 docs 文档对照

### 来自 `docs/Twistor-LNN-架构设计.md`:

```python
# 文档中的核心方程
dz/dt = (-z + W·tanh(z) + U·x) / τ(z)

# 实现 (twistor_100m_config.py)
def compute_dzdt(self, z, x):
    dz_real = -z_real + W_tanh_real + Ux + self.b_real
    dz_imag = -z_imag + W_tanh_imag + Ux + self.b_imag
    tau = self.compute_tau(z)
    dzdt = torch.complex(dz_real / tau, dz_imag / tau)
    return dzdt
```

### 来自 `docs/完成架构.md`:

```
架构流程:
  Input x(t) → Dynamics Core → Euler Integration → Output Decoder

实现:
  x_seq → TwistorLNNBlock → z + dt*dzdt → Linear(Re(z))
```

## 📈 参数量分析

### Large 配置 (~470M)

```
每层参数:
  - W_real: 2048×2048 = 4.2M
  - W_imag: 2048×2048 = 4.2M
```
