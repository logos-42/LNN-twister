## 🔧 使用方式

### 命令行

```bash
# 1. 基本用法 (本地提交)
python auto_archive.py -m "v0.4.1 完成"

# 2. 运行测试后提交
python auto_archive.py -t -m "v0.4.1 完成"

# 3. 运行测试并推送到 GitHub
python auto_archive.py -t -p -m "v0.4.1 完成"

# 4. 查看状态
python auto_archive.py --status

# 5. 回滚
python auto_archive.py --rollback
```

### GitHub Actions

推送到 main/master 分支时自动触发：

```yaml
push → GitHub Actions → 测试 → 通过 → 存档
                              ↓
                           失败 → 回滚
```
