## 1. 设计决策

### 1.1 核心定位

> **我们做的不是"严格扭量理论"，而是"扭量启发的复数动力系统"**

| 概念 | 严格版本 | 我们的版本 |
|-----|---------|-----------|
| 扭量结构 | Penrose 几何 | 复数向量 + 几何解释 |
| 时间积分 | ODE Solver | Euler（先）→ ODE（后） |
| 状态空间 | 严格 Twistor Space | ℂⁿ（复数向量） |
| 约束 | 完整几何约束 | 稳定性约束 |

**为什么这样选？**
- ✅ 可实现、可调试
- ✅ 保留核心思想（复数 + 动力学）
- ✅ 为未来扩展留空间

---

### 1.2 关键设计选择

```text
状态：复数向量 z ∈ ℂⁿ（twistor-inspired）
动力学：LNN + 稳定项
积分：Euler（dt=0.1）
τ：f(z.real) - 可学习时间常数
输出：从 z.real 解码
Loss：MSE + 稳定性约束
Dim：hidden=16（小模型）
Tensor：第一阶段不做
任务：时间序列预测
目标：先跑通，再优化
```

## 3. 数学定义

### 3.1 核心方程

#### 动力学方程

$$\frac{dz}{dt} = \frac{-z + W_z \cdot \tanh(z) + W_x \cdot x}{\tau(z)}$$

其中：
- $z \in \mathbb{C}^n$：复数状态
- $x \in \mathbb{R}^n$：输入
- $\tau(z) = \sigma(W_\tau \cdot \text{Re}(z)) + \epsilon$：时间常数

#### 分量形式（实部/虚部）

$$\frac{d\text{Re}(z)}{dt} = \frac{-\text{Re}(z) + \tanh(W_z \cdot \text{Re}(z)) + W_x \cdot x}{\tau(z)}$$

$$\frac{d\text{Im}(z)}{dt} = \frac{-\text{Im}(z) + \tanh(W_z \cdot \text{Im}(z))}{\tau(z)}$$

#### 输出方程

$$y = W_{out} \cdot \text{Re}(z) + b_{out}$$

---

### 3.2 时间常数 τ 的物理意义

| τ 值 | 变化速度 | 行为特征 |
|-----|---------|---------|
| τ ≈ 0.1 | 快 | 反应灵敏，易波动 |
| τ ≈ 1.0 | 中等 | 平衡 |
| τ ≈ 10.0 | 慢 | 反应迟钝，稳定 |
| τ 可变 | 自适应 | **"液态"特性** |

**关键**：τ 是状态的函数，不是固定值

---

### 3.3 数值积分（Euler）

$$z(t + \Delta t) = z(t) + \Delta t \cdot \frac{dz}{dt}$$

离散形式：

```python
z_new = z_old + dt * dz_dt
```

### 4.2 完整实现代码

```python
import torch
import torch.nn as nn
import torch.nn.functional as F


class TwistorLNN(nn.Module):
    def __init__(self, input_dim: int, hidden_dim: int, output_dim: int):
        super().__init__()
        self.hidden_dim = hidden_dim
        
        # 动力学参数
        self.Wz = nn.Linear(hidden_dim, hidden_dim)
        self.Wx = nn.Linear(input_dim, hidden_dim)
        self.W_tau = nn.Linear(hidden_dim, hidden_dim)
        
        # 输出层
        self.out = nn.Linear(hidden_dim, output_dim)
        
        # 初始化
        self._init_weights()
    
    def _init_weights(self):
        nn.init.xavier_normal_(self.Wz.weight)
        nn.init.xavier_normal_(self.Wx.weight)
        nn.init.constant_(self.W_tau.weight, 0.0)
        nn.init.constant_(self.W_tau.bias, 0.0)
    
    def forward(self, x_seq, dt=0.1):
        """
        Args:
            x_seq: [seq_len, batch, input_dim]
            dt: 时间步长
        
        Returns:
            outputs: [seq_len, batch, output_dim]
        """
        T, B, _ = x_seq.shape
        
        # 初始化复数状态
        z_real = torch.zeros(B, self.hidden_dim, device=x_seq.device)
        z_imag = torch.zeros(B, self.hidden_dim, device=x_seq.device)
        
        outputs = []
        z_trajectory = []
        
        for t in range(T):
            x = x_seq[t]
            
            # 计算时间常数
            tau = torch.sigmoid(self.W_tau(z_real)) + 1e-3
            
            # 计算动力学（实部/虚部分开）
            dz_real = (-z_real + torch.tanh(self.Wz(z_real)) + self.Wx(x)) / tau
            dz_imag = (-z_imag + torch.tanh(self.Wz(z_imag))) / tau
            
            # Euler 更新
            z_real = z_real + dt * dz_real
            z_imag = z_imag + dt * dz_imag
            
            # 记录轨迹（可选）
            z_trajectory.append(torch.complex(z_real, z_imag))
            
            # 输出（只用实部）
            y = self.out(z_real)
            outputs.append(y)
        
        return torch.stack(outputs), torch.stack(z_trajectory)
    
    def step(self, z, x, dt=0.1):
        """
        单步演化（用于智能体/强化学习）
        
        Args:
            z: [batch, hidden_dim], complex
            x: [batch, input_dim], real
            dt: 时间步长
        
        Returns:
            z_new: [batch, hidden_dim], complex
            output: [batch, output_dim]
        """
        z_real, z_imag = z.real, z.imag
        
        tau = torch.sigmoid(self.W_tau(z_real)) + 1e-3
        dz_real = (-z_real + torch.tanh(self.Wz(z_real)) + self.Wx(x)) / tau
        dz_imag = (-z_imag + torch.tanh(self.Wz(z_imag))) / tau
        
        z_new = torch.complex(
            z_real + dt * dz_real,
            z_imag + dt * dz_imag
        )
        
        output = self.out(z_new.real)
        return z_new, output
```

### 4.3 损失函数

```python
def twistor_loss(pred, target, z_real, dz_real, 
                 lambda_mse=1.0, lambda_stability=0.01):
    """
    复合损失函数
    
    Args:
        pred: 预测输出
        target: 目标值
        z_real: 实部状态
        dz_real: 状态变化率
        lambda_*: 权重
    """
    # 任务损失（MSE）
    loss_mse = ((pred - target) ** 2).mean()
    
    # 稳定性约束（防止爆炸）
    loss_stability = (dz_real ** 2).mean()
    
    # 能量约束（可选）
    loss_energy = (z_real ** 2).mean()
    
    # 总损失
    loss_total = (
        lambda_mse * loss_mse +
        lambda_stability * loss_stability +
        0.001 * loss_energy
    )
    
    return loss_total, {
        'mse': loss_mse.item(),
        'stability': loss_stability.item(),
        'energy': loss_energy.item()
    }
```

## 5. 训练方法

### 5.1 训练流程

```python
def train(model, dataloader, num_epochs=100, lr=1e-3):
    optimizer = torch.optim.Adam(model.parameters(), lr=lr)
    scheduler = torch.optim.lr_scheduler.ReduceLROnPlateau(
        optimizer, mode='min', factor=0.5, patience=20
    )
    
    for epoch in range(num_epochs):
        model.train()
        epoch_loss = 0
        
        for x_batch, y_batch in dataloader:
            optimizer.zero_grad()
            
            # 前向传播
            pred, z_traj = model(x_batch, dt=0.1)
            
            # 计算损失
            z_real = z_traj.real
            dz_real = torch.diff(z_real, dim=0).mean(dim=0)
            loss, _ = twistor_loss(pred, y_batch, z_real, dz_real)
            
            # 反向传播
            loss.backward()
            
            # 梯度裁剪（重要！）
            torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
            
            optimizer.step()
            epoch_loss += loss.item()
        
        avg_loss = epoch_loss / len(dataloader)
        scheduler.step(avg_loss)
        
        if (epoch + 1) % 10 == 0:
            print(f"Epoch {epoch+1}/{num_epochs}, Loss: {avg_loss:.4f}")
```

```python
class AdvancedTwistorDecoder(nn.Module):
    def __init__(self, hidden_dim, output_dim):
        super().__init__()
        self.vector_decoder = nn.Linear(hidden_dim, output_dim)
        self.tensor_decoder = nn.Linear(hidden_dim ** 2, output_dim)
    
    def forward(self, z, mode='vector'):
        if mode == 'vector':
            return self.vector_decoder(z.real)
        
        elif mode == 'tensor':
            # z → vector → tensor
            v = z.real
            tensor = torch.einsum('bi,bj->bij', v, v)
            tensor_flat = tensor.view(tensor.size(0), -1)
            return self.tensor_decoder(tensor_flat)
```

### 7.3 智能体接口

```python
class TwistorAgent:
    """
    基于 Twistor-LNN 的智能体
    """
    def __init__(self, obs_dim, action_dim, hidden_dim=32):
        self.model = TwistorLNN(obs_dim, hidden_dim, action_dim)
        self.z = None
    
    def reset(self):
        self.z = torch.zeros(1, self.model.hidden_dim, dtype=torch.cfloat)
    
    def act(self, obs, dt=0.1):
        if self.z is None:
            self.reset()
        
        x = torch.from_numpy(obs).float().unsqueeze(0)
        
        with torch.no_grad():
            self.z, action = self.model.step(self.z, x, dt)
        
        return action.numpy().squeeze(0)
```

## 8. 常见问题解答

### Q1: 为什么用复数而不是实数？

**A**: 复数天然支持：
- 相位（旋转）
- 振荡行为
- 几何变换

这些在实数空间需要额外参数。

### Q2: 时间常数 τ 怎么初始化？

**A**: 
```python
nn.init.constant_(W_tau.weight, 0.0)
nn.init.constant_(W_tau.bias, 0.0)
# 这样 τ ≈ sigmoid(0) + ε ≈ 0.5 + ε
```

### Q3: 如何调试动力学系统？

**A**:
1. 可视化相空间轨迹
2. 检查 ||dz/dt|| 是否爆炸
3. 减小 dt 观察稳定性
4. 添加梯度裁剪

### Q4: 这个架构能做什么任务？

**A**:
- ✅ 时间序列预测
- ✅ 系统辨识
- ✅ 简单控制（CartPole）
- ✅ 具身智能（需要扩展）
- ❌ 图像分类（不适合）
- ❌ NLP（需要修改）

### Q5: 如何扩展到更大模型？

**A**:
1. 增加 hidden_dim（16 → 64 → 128）
2. 使用 RK4 或 ODE 积分器
3. 添加多层 Twistor-LNN
4. 引入注意力机制

## 9. 总结

### 核心创新

1. **复数状态空间**: z ∈ ℂⁿ 替代 h ∈ ℝⁿ
2. **可学习时间常数**: τ(z) 实现液态特性
3. **动力学驱动**: 学习 dZ/dt 而非静态映射
4. **可解释性**: 相位、模长、轨迹都有几何意义

### 与 LNN 的关系

```
标准 LNN:     dh/dt = f(h, x),  h ∈ ℝⁿ
Twistor-LNN:  dZ/dt = F(Z, x),  Z ∈ ℂⁿ
```
