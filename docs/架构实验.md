# 四、“液态”的含义（你问的重点）

关键在这个：

## 👉 时间常数 τ（tau）

\tau(h) \frac{dh}{dt} = -h + W\sigma(h) + Ux

---

## τ 是什么？

一句话：

> **控制变化速度的“阻力/惯性”**

---

## 类比

* τ 小 → 反应快（神经元“兴奋”）
* τ 大 → 反应慢（神经元“迟钝”）

---

## 为什么叫“liquid”

👉 因为 τ 会变：

```text
不同状态 → 不同速度
```

像液体一样流动

# 八、动力学怎么写（你可以直接用）

最小版本👇

```python
def dz_dt(z, x):
    tau = sigmoid(W_tau(z.real))

    dz = (-z + Wz * tanh(z) + Wx * x) / tau

    return dz
```
