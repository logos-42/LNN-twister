## ✅ 0.2.1 新增功能

### 1. 自生产数据 ✅

```python
model = SelfTrainingTwistorLNN(input_dim=2, hidden_dim=32)

# 自动生成正弦波数据
X, y = model.generate_data(task_type='sine', n_samples=100)

# 自动生成 Lorenz 数据
X, y = model.generate_data(task_type='lorenz', n_samples=50)

# 自动生成自定义数据
X, y = model.generate_data(task_type='custom', func=my_func)
```

**支持的数据类型**:
- ✅ 正弦波 (sine)
- ✅ Lorenz 吸引子 (lorenz)
- ✅ 自定义函数 (custom)

---

### 2. 自训练循环 ✅

```python
result = model.self_training_loop(
    n_iterations=10,
    epochs_per_iteration=50,
    task_type='sine',
    verbose=True,
)
```

**工作流程**:
```
生成数据 → 训练模型 → 评估性能 → 调整策略 → (循环)
```

**测试结果**:
```
============================================================
自训练循环开始
============================================================
迭代 1/10: 验证 MSE=0.080 (目标：0.01) ✅ 达到目标！提前终止
============================================================
自训练循环完成
============================================================
```

---

### 3. 性能评估 ✅

```python
metrics = model.evaluate_performance(X_val, y_val)

# 返回指标:
# - MSE: 均方误差
# - MAE: 平均绝对误差
# - SNR: 信噪比 (dB)
# - Loss: 用于策略调整
```

---

### 4. 策略调整 ✅

```python
# 根据性能自动调整数据生成策略
model.adjust_data_strategy(metrics)

# 调整逻辑:
# - 损失高 → 增加数据量，降低噪声
# - 损失中等 → 适度增加数据
# - 损失达标 → 保持或减少数据
```

---

### 5. 课程学习 ✅

```python
trainer = AutoCurriculumTrainer(model)
trainer.define_curriculum()  # 使用默认课程

result = trainer.train_with_curriculum(
    epochs_per_stage=100,
    verbose=True,
)
```

**默认课程**:
1. 简单正弦波 (freq: 0.5-1.0, noise: 0.05)
2. 中等正弦波 (freq: 0.5-2.0, noise: 0.10)
3. 复杂正弦波 (freq: 1.0-3.0, noise: 0.15)
4. Lorenz 吸引子 (混沌系统)

## 📝 使用示例

### 快速开始 (5 行代码)

```python
from twistor_lnn import SelfTrainingTwistorLNN

model = SelfTrainingTwistorLNN(input_dim=2, hidden_dim=32)
result = model.self_training_loop(n_iterations=10)
print(f"最终 MSE: {result['final_metrics']['mse']}")
```

### 课程学习 (10 行代码)

```python
from twistor_lnn import SelfTrainingTwistorLNN, AutoCurriculumTrainer

model = SelfTrainingTwistorLNN(input_dim=2, hidden_dim=32)
trainer = AutoCurriculumTrainer(model)
trainer.define_curriculum()
result = trainer.train_with_curriculum()
print(f"完成阶段：{sum(result['completed'])}/{trainer.n_stages}")
```
