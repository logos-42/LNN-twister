## 🎯 核心架构 (基于 docs)

### 动力学方程

```
dz/dt = (-z + W·tanh(z) + U·x + b) / τ(z)

其中:
  - z ∈ ℂⁿ 是复数状态
  - τ(z) = sigmoid(W_τ·|z|) + ε 是状态依赖时间常数
  - W_real, W_imag 分别处理实部和虚部
```

### 架构流程

```
Input x(t) → Dynamics Core → Euler Integration → Output Decoder
           dZ/dt = F(Z, x)   z += dt*dz/dt      y = Linear(Re(z))
```

## 🔧 使用方式

### 1. 基础使用 (v0.2.1)

```python
from twistor_lnn import TwistorLNN

model = TwistorLNN(input_dim=2, hidden_dim=32, output_dim=1)
x = torch.randn(50, 1, 2)
y = model(x)
```

### 2. 大规模架构 (v0.3.0)

```python
from twistor_100m_config import StackedTwistorLNN, TwistorLNNConfig

config = TwistorLNNConfig(hidden_dim=1024, n_layers=10)
model = StackedTwistorLNN(config)
```

### 3. 自动化进化 (v0.4.0)

```python
from auto_evolution import AutoEvolutionSystem

system = AutoEvolutionSystem(StackedTwistorLNN)
best = system.run_evolution(n_generations=10)
config = system.get_best_config()
```

### 是什么

> **Twistor-LNN 是基于扭量理论的液态神经网络**
> 
> 专为连续时间序列设计
> 
> 具有自动进化能力

### 不是什么

> **不是 Transformer**
> 
> **不是语言模型**
> 
> **不是 LLM 的替代品**
