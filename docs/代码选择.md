# 二、为什么不是“只用 Python”

你现在大概率在用 Python，对吧。

## Python 的位置（说实话）

优点：

* AI 生态最强（PyTorch 等）
* 试验快
* 论文复现快

缺点（致命）：

* 慢（尤其是你做 ODE / 动力系统）
* 不适合复杂系统架构
* 并发差
* 难做长期工程

👉 结论：

> Python 适合“验证想法”，不适合“承载系统”

# 三、为什么 Rust 是你这个方向的“长期解”
