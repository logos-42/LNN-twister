## 🚀 使用方式

### 前提条件

```bash
# 1. 安装 Ollama
# macOS: brew install ollama
# Windows: 从 https://ollama.ai 下载
# Linux: curl -fsSL https://ollama.ai/install.sh | sh

# 2. 下载模型
ollama pull llama2

# 3. 启动 Ollama
ollama serve
```

### 快速开始

```bash
# 运行测试
python test_karpov.py

# 运行 3 次迭代
python karpov_cycle.py --iterations 3

# 仅分析代码
python karpov_cycle.py --analyze-only

# 使用 CodeLlama
python karpov_cycle.py --model codellama --iterations 5
```
