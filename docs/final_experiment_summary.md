## 关键发现

### 1. 流形约束权重的有效性
从分离权重到流形约束权重，PPL从141降到24.55（↓83%）。
几何约束不是装饰，而是**内生于权重生成过程**。

### 2. 正则化强度的权衡
| 正则化权重 | PPL | 稀疏化 | 秩序指数 |
|-----------|-----|--------|---------|
| 0 (无) | ~25 | ~2% | ~0.3 |
| 0.001 (低) | 24.2 | 1.8% | 0.305 |
| 0.005 (高) | 22.8 | 62% | 0.486 |
| 0.01 (过高) | ~30 | 98% | ~0.35 |

**最优正则化权重在 0.003-0.005 之间**。

### 3. 真实n-gram评估
- 2-gram覆盖率 88.5% 说明模型学到了**英文字符对的基本统计规律**
- 3-gram覆盖率 53.1% 说明**部分三字符组合是有效的英文模式**
- 高多样性(94-100%)说明模型没有过度拟合到固定模式

### 4. 神经元固化
所有实验都显示**44-62%的神经元完全固化 (cons=1.0)**。
固化机制有效工作，保护已学习表示。

## 架构总结

### 流形约束权重
```
W = A · exp(i·Θ)

A ∈ ℝ^(N×N)  ← 振幅(自由学习, 受L1/L2正则化)
θ ∈ ℝ^(N×3)  ← 流形坐标(可学习)
Θ_ij = twist(θ_i, θ_j)  ← 相位(莫比乌斯几何内生决定)
```

### 优势
1. **几何内生约束**: 相位由流形决定，不是事后投影
2. **全局统一**: 所有权重都受约束
3. **旧权重也被扰动**: θ优化时所有相位都变
4. **不破坏现有网络**: 振幅仍可自由学习

### 局限
1. 振幅仍然是自由参数（N×N个）
2. 相位初始化均匀分布，缺乏结构化引导
3. 语义连贯性仍然不足

## 下一步方向

1. **结构化相位初始化**: 基于验证集n-gram统计初始化相位
2. **振幅分组正则化**: 对不同连接类型使用不同正则化强度
3. **更长训练**: 10000-20000步观察PPL是否继续下降
4. **多尺度流形**: 不同层使用不同流形维度
5. **语义评估**: 使用词级别n-gram和困惑度

## 结论

流形约束权重（振幅+相位）是当前架构最有效的改进：
- PPL 从 449 降到 22-25（↓95%）

这验证了核心假设：**几何约束生长促进秩序涌现**。
