## 📊 架构规格

### 核心方程

```
dz/dt = (-z + W·tanh(z) + U·x) / τ(z)

其中:
  - z ∈ ℂⁿ 是复数状态
  - τ(z) = sigmoid(W_τ·|z|) 是状态依赖时间常数
  - W, U 是液态参数
```

| 配置 | Small | Medium | Large (100M) |

| **hidden_dim** | 128 | 512 | 2048 |
| **n_layers** | 4 | 8 | 16 |
| **参数量** | 267K | 8.4M | **269M** |
| **输入** | 连续值 | 连续值 | 连续值 |
| **输出** | 连续值 | 连续值 | 连续值 |

## 🔧 使用方法

### 1. 创建模型

```python
from twistor_liquid_100m import LiquidTwistorLNN, LiquidTwistorConfig

# 大规模配置 (~269M 参数)
config = LiquidTwistorConfig.large_100m()
model = LiquidTwistorLNN(config)

# 或小规模测试
config = LiquidTwistorConfig.small()
model = LiquidTwistorLNN(config)
```

### 2. 时间序列预测

```python
import torch

# 输入：时间序列 (T, B, input_dim)
x = torch.randn(100, 1, 1)  # 100 步，1 个样本，1 维

# 预测
with torch.no_grad():
    y = model(x)

print(f"输出形状：{y.shape}")  # (100, 1, 1)
```

### 3. 多步预测

```python
# 根据过去 50 步预测未来 10 步
past = torch.randn(50, 1, 1)
future = torch.randn(10, 1, 1)

# 训练
x = torch.cat([past, future], dim=0)
y_pred = model(x)
loss = F.mse_loss(y_pred[:50], past)  # 重构损失
```

### 4. 训练


## 📈 参数量分析

### Large 配置 (~269M)

```
每层参数:
  - W_real: 2048 × 2048 = 4.2M
  - W_imag: 2048 × 2048 = 4.2M
  - W_tau:  2048 × 2048 = 4.2M
  - U:      2048 × 2048 = 4.2M
  - 其他：~1M
  
  每层总计：~17M

16 层总计：~272M
```

### 调整到 100M

```python
@classmethod
def large_100m(cls):
    return cls(
        hidden_dim=1024,      # 从 2048 降至 1024
        n_layers=12,          # 从 16 降至 12
    )
# 新参数量：~80M
```
