## 🧬 进化算法

### 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| **population_size** | 20 | 种群大小 |
| **elite_size** | 4 | 精英数量 |
| **mutation_rate** | 0.3 | 变异率 |

### 进化流程

```
Generation 0:
  1. 初始化种群 (随机采样)
  2. 评估所有个体
  3. 选择精英
  4. 交叉 + 变异 → 新一代

Generation 1:
  1. 评估所有个体
  2. 选择精英
  3. 交叉 + 变异 → 新一代

...

Generation N:
  1. 评估所有个体
  2. 返回最优个体
```

### 适应度计算

```python
fitness = Σ (weight_i × score_i)

其中:
  - stability 权重最高 (2.0)
  - 预测任务权重中等 (1.0)
  - 速度权重较低 (0.5)
```

### 示例 3: 多目标优化

# 自定义适应度函数 (强调稳定性)
def stability_focused_fitness(results):
    fitness = 0.0
    
    # 稳定性权重最高
    if 'stability' in results:
        fitness += 5.0 * results['stability'].metric_value
    
    # 预测任务
    if 'sine_forecast' in results:
        fitness += 1.0 / (1.0 + results['sine_forecast'].metric_value)
    
    # 速度
    if 'inference_speed' in results:
        fitness += 0.5 * (1.0 - results['inference_speed'].metric_value)
    
    return fitness

# 应用自定义适应度
system.optimizer._compute_fitness = stability_focused_fitness

# 运行进化
best = system.run_evolution(n_generations=10)

### 2. 早停策略

```python
class EarlyStoppingEvolution(EvolutionaryOptimizer):
    def __init__(self, *args, patience=5, min_improvement=0.01):
        super().__init__(*args)
        self.patience = patience
        self.min_improvement = min_improvement
        self.no_improve_count = 0
    
    def evolve(self, *args, **kwargs):
        for gen in range(kwargs['n_generations']):
            # ... 进化逻辑
            
            # 检查改进
            if gen > 0:
                improvement = best_fitness - prev_best_fitness
                if improvement < self.min_improvement:
                    self.no_improve_count += 1
                else:
                    self.no_improve_count = 0
                
                if self.no_improve_count >= self.patience:
                    print(f"早停于第 {gen} 代")
                    break
            
            prev_best_fitness = best_fitness
```
