# 流形约束权重大规模实验记录

## 实验日期
2026-04-05

## 实验目的
验证振幅+相位(流形约束)权重表示是否促进秩序涌现。

## 架构变更

### 旧权重表示
```
W_real ∈ ℝ^(N×N)  ← 自由参数
W_imag ∈ ℝ^(N×N)  ← 自由参数
```

### 新权重表示 (流形约束)
```
A ∈ ℝ^(N×N)  ← 振幅(自由学习)
θ ∈ ℝ^(N×3)  ← 流形坐标(可学习)
Θ_ij = twist(θ_i, θ_j)  ← 相位(莫比乌斯几何决定)

W = A · exp(i·Θ)  ← 复数权重由振幅+相位生成
```

**关键区别**: 相位不是自由参数，而是由神经元在流形上的位置决定。

### 新 (流形约束)


**观察**:
- 流形约束版本的空格分布更合理
- 出现了更多标点符号的模式化使用
- 类单词结构更明显 (`hfuotrcnaotm`, `ritirnuenfa`)
- 但语义连贯性仍需改进

## 关键发现

### 1. 几何约束生效
98.6% 的权重振幅接近零，说明流形约束有效抑制了自由参数。
只有 1.4% 的连接有显著振幅，形成了**稀疏核心网络**。

### 2. 相位均匀分布
相位标准差 ≈ 2.0 (接近均匀分布 [0, 2π))，说明莫比乌斯扭转张量
在引导相位分布方面有效。

### 3. 秩序指数大幅提升
0.347 → 0.596，从"混沌边缘"进入"有序区域"。
但仍未达到 0.8+ 的"高度有序"状态。

### 4. 训练效率提升
Loss 改善从 18% 提升到 48%，PPL 从 141 降到 24.55。
流形约束减少了有效参数数量，优化更高效。

## 下一步

1. **更长训练**: 5000-10000 步观察秩序是否继续涌现
2. **相位初始化优化**: 当前相位均匀分布，可能需要更结构化的初始化
3. **振幅正则化**: 鼓励更多权重趋零，进一步稀疏化
4. **语义评估**: 使用 n-gram 匹配度量化语义连贯性

## 结论

振幅+相位(流形约束)权重表示显著促进了秩序涌现。
几何约束不是装饰，而是**内生于权重生成过程**，
使得 98.6% 的参数自发稀疏化，同时提升了训练效率和文本质量。
