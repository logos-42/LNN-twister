## 🔧 使用方法

### 1. 创建模型

```python
from twistor_llm_100m import TwistorLNN_100M, TwistorLNNConfig

# 创建 100M 配置
config = TwistorLNNConfig.large_100m()
model = TwistorLNN_100M(config)

# 或创建小模型用于测试
config = TwistorLNNConfig.small()
model = TwistorLNN_100M(config)
```

### 2. 推理

```python
import torch

# 输入
input_ids = torch.randint(0, 32000, (1, 32))

# 前向传播
with torch.no_grad():
    logits = model(input_ids)

print(f"输出形状：{logits.shape}")
# (batch, seq_len, vocab_size)
```

### 3. 文本生成

```python
# 自回归生成
input_ids = torch.tensor([[1, 2, 3, 4, 5]])

generated = model.generate(
    input_ids,
    max_new_tokens=100,
    temperature=0.8,
    top_k=50,
)

print(f"生成形状：{generated.shape}")
```

### 4. 训练

```bash
# 小规模测试
python train_100m.py

# 修改配置 (在 train_100m.py 中)
config = TrainingConfig(
    model_size="small",  # small, medium, large
    batch_size=32,
    learning_rate=1e-4,
    total_steps=100000,
)
```

## 🔍 架构调整建议

### 当前问题

实际参数量远超设计目标：
- Medium: 设计 50M → 实际 222M (4.4x)
- Large: 设计 100M → 实际 1.27B (12.7x)

### 调整方案

```python
# 调整后的 Medium 配置 (~50M)
@classmethod
def medium(cls):
    return cls(
        vocab_size=32000,
        hidden_dim=512,      # 从 1024 降至 512
        intermediate_dim=2048,  # 从 4096 降至 2048
        n_layers=12,
        n_heads=8,
        n_kv_heads=4,
    )

# 调整后的 Large 配置 (~100M)
@classmethod
def large_100m(cls):
    return cls(
        vocab_size=32000,
        hidden_dim=768,      # 从 2048 降至 768
        intermediate_dim=3072,  # 从 8192 降至 3072
        n_layers=16,         # 从 24 降至 16
        n_heads=12,
        n_kv_heads=4,
    )
```
