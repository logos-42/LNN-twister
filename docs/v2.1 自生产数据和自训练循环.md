## 📋 概述

v2.1 为 Twistor-LNN 添加了**自生产数据、自训练循环**的能力，实现：

1. **自动生成训练数据** - 无需手动准备数据集
2. **自动训练循环** - 生成→训练→评估→调整全自动
3. **性能评估** - MSE、MAE、SNR 多指标评估
4. **策略调整** - 根据性能自动调整数据生成策略
5. **课程学习** - 从简单到复杂逐步提升

## 1. SelfTrainingTwistorLNN 类

```

│  ┌─────────────────────────────────┐   │

```

### 使用示例

```python
from twistor_lnn import SelfTrainingTwistorLNN

# 创建模型
model = SelfTrainingTwistorLNN(
    input_dim=2,

    output_dim=1,

result = model.self_training_loop(

    verbose=True,
)

# 查看结果
print(f"最终 MSE: {result['final_metrics']['mse']}")
```

## 2. 数据生成功能

```python
# 1. 正弦波数据
X, y = model.generate_data(task_type='sine', n_samples=100, seq_len=50)

# 2. Lorenz 吸引子数据
X, y = model.generate_data(task_type='lorenz', n_samples=50, seq_len=50)

# 3. 自定义数据

```

### 数据生成参数

```python
model.data_params = {
    'freq_range': (0.5, 2.0),    # 频率范围
    'noise_std': 0.1,             # 噪声标准差
    'seq_len': 50,                # 序列长度
    'n_samples': 100,             # 样本数量
}
```

### 自定义数据生成

```python
# 定义自定义函数
def my_func(t, params):
    a, b, c = params
    return a * np.exp(-b * t) * np.sin(c * t)

# 生成数据
X, y = model.generate_data(
    task_type='custom',
    n_samples=100,
    seq_len=50,
    func=my_func,
)
```

## 3. 自训练循环

### 工作流程

```
迭代 1:
  ┌──────────┐
  │ 生成数据  │ → X_train, y_train
  └─────┬────┘
        │
  ┌─────▼────┐
  │ 训练模型  │ → 优化参数
  └─────┬────┘
        │
  ┌─────▼────┐
  │ 评估性能  │ → MSE, SNR
  └─────┬────┘
        │
  ┌─────▼────┐
  │ 调整策略  │ → 更新 data_params
  └─────┬────┘
        │
        v
迭代 2: (使用新参数生成数据)
  ...
```

### 完整示例

```python
model = SelfTrainingTwistorLNN(input_dim=2, hidden_dim=16)

# 设置性能目标
model.performance_metrics = {
    'target_loss': 0.01,    # 目标损失
    'min_samples': 50,       # 最小样本数
    'max_samples': 500,      # 最大样本数
    'patience': 10,          # 早停耐心值
}

# 启动自训练
result = model.self_training_loop(
    n_iterations=10,
    epochs_per_iteration=50,
    task_type='sine',
    device='cuda',  # 使用 GPU
    verbose=True,
)

# 分析结果
history = result['history']
print(f"训练损失曲线：{history['train_loss']}")
print(f"验证损失曲线：{history['val_loss']}")
```

## 4. 策略调整机制

### 调整逻辑

```python
if current_loss > target_loss * 10:
    # 损失远高于目标：大幅增加数据，降低噪声
    n_samples = min(n_samples * 2, max_samples)
    noise_std = max(noise_std * 0.8, 0.01)

elif current_loss > target_loss:
    # 损失高于目标：适度增加数据
    n_samples = min(int(n_samples * 1.2), max_samples)

else:
    # 损失达到目标：保持或减少数据
    n_samples = max(int(n_samples * 0.9), min_samples)
```

### 性能指标

| 指标 | 计算方式 | 说明 |
|------|---------|------|
| MSE | `mean((y_pred - y_true)²)` | 均方误差 |
| MAE | `mean(|y_pred - y_true|)` | 平均绝对误差 |
| SNR | `10 * log10(signal_power / noise_power)` | 信噪比 (dB) |
| Loss | MSE | 用于策略调整 |

## 5. 课程学习 (Curriculum Learning)

### 核心思想

```
从简单任务开始 → 逐步增加难度 → 达到复杂任务

阶段 1: 简单正弦波 (freq: 0.5-1.0, noise: 0.05)
  ↓ 完成
阶段 2: 中等正弦波 (freq: 0.5-2.0, noise: 0.1)
  ↓ 完成
阶段 3: 复杂正弦波 (freq: 1.0-3.0, noise: 0.15)
  ↓ 完成
阶段 4: Lorenz 吸引子 (混沌系统)
```

### 使用示例

```python
from twistor_lnn import SelfTrainingTwistorLNN, AutoCurriculumTrainer

# 创建模型
model = SelfTrainingTwistorLNN(input_dim=2, hidden_dim=32)

# 创建课程训练器
trainer = AutoCurriculumTrainer(model)

# 定义课程 (或使用默认课程)
trainer.define_curriculum(stages=[
    {
        'name': '简单正弦波',
        'task_type': 'sine',
        'freq_range': (0.5, 1.0),
        'noise_std': 0.05,
        'seq_len': 30,
        'target_loss': 0.1,
    },
    {
        'name': 'Lorenz 吸引子',
        'task_type': 'lorenz',
        'seq_len': 50,
        'target_loss': 0.05,
    },
])

# 开始课程学习
result = trainer.train_with_curriculum(
    epochs_per_stage=100,
    device='cuda',
    verbose=True,
)

# 查看结果
print(f"完成阶段：{sum(result['completed'])}/{len(result['stage'])}")
```

### 默认课程

```python
默认课程包含 4 个阶段:
  1. 简单正弦波 (freq: 0.5-1.0, noise: 0.05, seq_len: 30)
  2. 中等正弦波 (freq: 0.5-2.0, noise: 0.10, seq_len: 50)
  3. 复杂正弦波 (freq: 1.0-3.0, noise: 0.15, seq_len: 70)
  4. Lorenz 吸引子 (混沌系统)
```

## 7. API 参考

### SelfTrainingTwistorLNN

```python
class SelfTrainingTwistorLNN(nn.Module):
    """支持自生产数据和自训练循环的 Twistor-LNN"""
    
    def __init__(
        self,
        input_dim: int = 2,
        hidden_dim: int = 32,
        output_dim: int = 1,
        dt: float = 0.1,
        **kwargs
    )
    
    def generate_data(
        self,
        task_type: str = 'sine',
        n_samples: int = 100,
        seq_len: int = 50,
        **kwargs
    ) -> Tuple[torch.Tensor, torch.Tensor]
    
    def self_training_loop(
        self,
        n_iterations: int = 10,
        epochs_per_iteration: int = 50,
        task_type: str = 'sine',
        val_split: float = 0.2,
        device: str = 'cpu',
        verbose: bool = True,
    ) -> Dict
    
    def evaluate_performance(
        self,
        X_val: torch.Tensor,
        y_val: torch.Tensor
    ) -> Dict[str, float]
    
    def forward(self, x: torch.Tensor) -> torch.Tensor
```

### 故障排除

**问题 1: 训练不收敛**

```python
# 解决方案
```

