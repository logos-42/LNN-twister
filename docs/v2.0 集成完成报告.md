## 📊 API 使用示例

### 1. 多任务学习

```python
from twistor_lnn import MultiTaskTwistorLNN, TaskConfig

# 定义任务
task_configs = [
    TaskConfig(task_id=0, task_name="sine", input_dim=2, output_dim=1),
    TaskConfig(task_id=1, task_name="cosine", input_dim=2, output_dim=1),
]

# 创建模型
model = MultiTaskTwistorLNN(task_configs, hidden_dim=32)

# 多任务前向传播
y_sine = model(x_sine, task_name="sine")
y_cosine = model(x_cosine, task_name="cosine")

# 零样本迁移
y_transfer = model.zero_shot_transfer(
    x, source_task="sine", target_task="cosine"
)
```

### 2. 元学习 (MAML)

```python
from twistor_lnn import MetaTwistorLNN

# 创建元学习模型
meta_model = MetaTwistorLNN(input_dim=2, hidden_dim=32)

# 元训练
query_loss, adapted_params = meta_model.meta_update(
    x_support, y_support, x_query, y_query
)

# 零样本适应 (5 步梯度下降)
y_pred = meta_model.zero_shot_adapt(
    x_few_shot, y_few_shot, x_test,
    adapt_steps=5
)
```

### 3. 提示学习

```python
from twistor_lnn import PromptTwistorLNN

# 创建提示学习模型
prompt_model = PromptTwistorLNN(
    input_dim=2, hidden_dim=32,
    n_prompts=10, prompt_dim=8
)

# 零样本推理
y = prompt_model(x_new_task)
```

## 🎯 推荐使用场景

### 使用 MultiTaskTwistorLNN，如果:
- ✅ 有 3 个以上相关任务
- ✅ 任务之间有正向迁移潜力
- ✅ 需要降低部署成本

### 使用 MetaTwistorLNN，如果:
- ✅ 需要快速适应新任务
- ✅ 新任务只有少量样本 (5-10 个)
- ✅ 可以接受元训练成本

### 使用 PromptTwistorLNN，如果:
- ✅ 需要零样本迁移
- ✅ 任务可以用提示组合描述
- ✅ 需要最低训练成本

## 📝 训练建议

```python
# 推荐：交替训练任务
for epoch in range(n_epochs):
    for task in tasks:
        x, y = get_task_data(task)

        loss = F.mse_loss(y_pred, y)
```

## 🔧 故障排除

### 问题 1: 多任务训练不收敛

```python
# 解决方案 1: 增加任务嵌入维度
model = MultiTaskTwistorLNN(..., task_embedding_dim=16)

# 解决方案 2: 增加门控网络容量
self.task_gates[task_name] = nn.Sequential(
    nn.Linear(task_embedding_dim, hidden_dim * 2),
    nn.ReLU(),
    nn.Linear(hidden_dim * 2, hidden_dim),
    nn.Sigmoid()
)
```

### 问题 2: 元学习 OOM

```python
# 解决方案 1: 减少内层步数
model.meta_update(..., inner_steps=3)  # 从 5 减少到 3

# 解决方案 2: 减小 hidden_dim
model = MetaTwistorLNN(..., hidden_dim=16)
```
